O que são números binários e hexadecimais?

Os números binários são a base dos computadores, representando dados apenas com os dígitos 0 e 1. Cada bit representa uma potência de 2, e um conjunto de 8 bits forma um byte, usado para armazenar dados.

O sistema hexadecimal é utilizado para representar valores binários de forma mais compacta. Ele usa 16 símbolos: 0-9 e A-F, sendo muito comum na programação, como na definição de cores em CSS e na manipulação de endereços de memória.

Exemplo de conversão:
Decimal: 255 → Binário: 11111111 → Hexadecimal: FF
